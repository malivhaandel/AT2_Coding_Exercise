# creative coding 2026

a collection of weekly experiments exploring generative design, typography, image systems, and motion using p5.js.

this repository documents my progress through different creative coding techniques, focusing on how text, images, noise, and interaction can be transformed into dynamic visual systems.

---

## projects

### week 2 – rays and fades
an early experiment with light, motion, and fading trails. this sketch explores how repeated geometry and transparency can create depth and movement over time.

### week 3 – text to point
a typography system that converts text into point-based geometry using font sampling. the text becomes a visual structure made of particles and spatial relationships.

### week 4 – type block
a layout study focused on structured typography blocks. explores alignment, hierarchy, and how text behaves in constrained grids and compositions.

### week 4 – type string
an extension of typographic experimentation, focusing on string-based systems, randomness, and dynamic phrase generation.

### week 5 – image array
an image system using arrays to store, switch, and manipulate multiple visual assets. introduces cycling and interactive image-based composition.

### week – random noise
a generative sketch driven by noise functions. explores randomness, organic movement, and procedural texture generation.
